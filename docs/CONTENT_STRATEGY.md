# Content Strategy

## Website Goals

The website exists to:

1. Build trust immediately.
2. Showcase high-end filmmaking.
3. Position Eliya as a director, not just a videographer.
4. Generate high-quality inquiries.
5. Feel like an experience rather than a portfolio.

Every page should answer: *"Why should someone trust Eliya with their story?"*

## Portfolio Categories

- **Hotels & Resorts** — luxury hospitality, experiences, destinations.
- **Outdoor Brands** — adventure, equipment, apparel, action sports.
- **Commercials** — brand campaigns and product films.
- **Travel Films** — personal films and cinematic documentaries.

Each category has its own page; each project has its own case study (see [WEBSITE_ARCHITECTURE.md](WEBSITE_ARCHITECTURE.md)).

## Every Project Should Answer

- What was the challenge?
- What story was told?
- What emotions should the audience feel?
- How was it filmed?
- What was my role?

## SEO Checklist (every page)

- Unique title
- Unique description
- Open Graph image
- Structured data
- Canonical URL

Note: not yet implemented sitewide — no sitemap package, no per-page Open Graph tags or structured data beyond the title/description set in `BaseLayout.astro`. This is tracked as future work, not part of this docs cleanup.
