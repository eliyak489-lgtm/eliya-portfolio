# Tone of Voice

Always write like a filmmaker. Never write like a marketing agency.

## Avoid

- world class
- cutting edge
- innovative
- revolutionary
- passionate
- excellence

## Prefer

- clear
- simple
- confident
- human
- cinematic

Write short sentences. Avoid hype. Avoid buzzwords.

Communicate quality without saying quality.

## Example

**Bad:** "We create cutting-edge visual experiences."

**Good:** "We create films people remember."
